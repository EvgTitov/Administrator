Скрипты запускаются только от админа.


Если нужно вынести пользователю той или иной скрипт для постоянного пользования делаем так:
1.Отправляем на рабочий стол как ярлык
2.Заходим в свойства
3.Во вкладке "Ярлык" жмём "Дополнительно"
4.Ставим галку "Запуск от имени системного администратора"

В некоторых скриптах для их адекватной работы необходимо менять параметры подходящие под комп как пример "Перезагрузка сетевого адаптера"
1.Для изменения скрипта жмём ПКМ->Изменить
2.Ореинтируемся по комментариям в начале скрипта по центру
